Help

Basics

Click a tool in the top to select it.
Then, click in the chart to add it to the map

When you place a time modifier (BPM, Time Signature Changer, or Hi-Speed Change), it will prompt you for the updated info

The mouse tool will let you move notes

Saving

"Save SUS" (CTRL + ALT + S) will export to a file.
This file cannot be loaded back into this program

"Save MMW" (CTRL + S) will export to a file.
This file can be loaded back into this program, with "Load MMW" (CTRL + L)
